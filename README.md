# Thread Pool

## Brief description

Simple thread pool example.

## Building

This project has been developed on Ubuntu 18.04, but it should work on any platform. 

Build on Linux:

```
    mkdir build
    cd build
    cmake ..
    make
```

If you want to generate VS 2017 project:

```
    mkdir build
    cd build
    cmake "Visual Studio 15 2017 Win64" ..
```
